# 💳 Credit Card Sales Analysis

## 📌 Project Overview
This project analyzes credit card sales data using Python to understand spending behavior and transaction trends.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Analysis Performed
- Sales trend analysis
- Customer spending patterns
- Data visualization using charts

## 🚀 How to Run
1. Clone repository
2. Install libraries:
   pip install pandas matplotlib
3. Run Python file or notebook

## 📁 Files
- credit_card_sales_analysis.py OR notebook.ipynb

## 📌 Outcome
Helps understand financial data trends using Python.
